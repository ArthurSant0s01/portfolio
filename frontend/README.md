# Frontend (Vite + React)

This frontend is built with Vite and React.

## Available Scripts

In the project directory, you can run:

### `yarn dev`

Runs the app in development mode.

### `yarn build`

Builds the app for production into `dist/`.

### `yarn preview`

Serves the production build locally.

### Environment

Set `VITE_BACKEND_URL` to the backend origin (for example `https://api.example.com`).
